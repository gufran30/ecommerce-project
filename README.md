## 📌 About The Project

This is a **frontend e-commerce web application** built using **React.js** and **Vite** as the build tool. The project demonstrates core e-commerce functionalities such as browsing products, managing a shopping cart, and navigating through pages — all with a clean and responsive UI.

It is a practice/learning project developed to strengthen skills in React component architecture, state management, routing, and modern frontend tooling.

---

## ✨ Features

- 🏠 **Home Page** — Product listing with browsable catalog
- 🔍 **Product Pages** — Detailed view for individual products
- 🛒 **Shopping Cart** — Add, remove, and manage cart items
- 📱 **Responsive Design** — Works across desktop and mobile screens
- ⚡ **Fast Development** — Powered by Vite with Hot Module Replacement (HMR)
- 🎨 **Custom Styling** — Clean CSS-based UI design

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/gufran30/ecommerce-project.git
   ```

2. **Navigate into the project directory**

   ```bash
   cd ecommerce-project
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**

   ```bash
   npm run dev
   ```

5. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder, ready to deploy.
